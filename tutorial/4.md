# 函数

Shell中函数定义的格式如下：

    函数名 ()
    {
        语句
        return 返回值;
    }

函数的参数通过`$*`，`$@`，`$1`，`$2`等获取。

下面是一个例子：

    check_psw()
    {
        if [ "$1" = "123456" ]
        then
            return 0
        else
			return 1
		fi
    }

    read x
    if check_psw "$x"
    then
        echo "pass"
    else
        echo "not pass"
    fi

这个例子中`check_psw`函数判断参数是否是`123456`，`S1`用来获得第一个参数。

值得注意的是，返回值为0是真，为1是假。
